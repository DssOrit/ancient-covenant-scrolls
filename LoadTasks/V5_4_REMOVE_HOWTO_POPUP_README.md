# Load Tasks v5.4: Remove How-To Popup

## What this version does

This version removes the blocking full-screen How-To pop-up entirely.

## What stays

- Agent Lab
- Alert Dashboard
- Notes system
- Help Center
- Load AI Helper
- Validator
- Repair Command Center
- Repair Preview
- Project Vault
- Focus Mode
- Launch Certificate

## What changed

Step-by-step help is now inside the Help Center instead of a blocking pop-up.

## Files to replace

Replace these in the LoadTasks folder:

- index.html
- app.js
- styles.css
- service-worker.js
- manifest.json

## After upload

1. Wait 2 to 5 minutes.
2. Open the site.
3. Refresh.
4. Confirm no How-To pop-up blocks the page.
5. Confirm Dashboard opens.
6. Confirm Agent Lab and Notes are still visible.
