Color Contrast Checker

How it works

WCAG 2.1 AA / AAA contrast ratio

Pick a foreground and background color. The tool computes the WCAG relative-luminance contrast ratio and tells you whether it passes for normal text, large text, and UI / graphical objects. If it fails, the suggestion line gives the smallest color tweak that would pass.
