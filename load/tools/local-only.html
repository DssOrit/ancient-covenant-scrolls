Local-only Mode

BUILD MARKER: LOCAL-ONLY-v1-2026-05-07-v17g43
Reliability addendum #7. When Local-only mode is on, every cloud / free-api route is blocked at the fetch boundary via LoadLocalOnly.guardedFetch(). Only built-in, local, prompt-only, and user-imported provider types remain available. State persists in LoadDB.kv['local-only']. The Chat Studio toolbar reflects this toggle.
OFF

Allowed when on

Blocked when on
