Reel Composer

How it works

Scene cards + images + audio → one WebM video

Drop the scene-cards JSON from Verse to Video. Optionally attach a per-scene image and audio take. Tap Render reel — each scene plays its image with Ken-Burns motion + verse text overlay + your audio. The browser captures the canvas via MediaRecorder and saves a WebM.

1. Scene cards JSON

Drop verse-to-video-scenes-*.json or any scene-cards JSON Expects { scenes: [{ verse, durationSeconds, aspect, ... }] } shape.

2. Per-scene assets

Tap Image to attach a per-scene image, Audio to attach a take. Scenes without images use a gradient placeholder; without audio play silent.
